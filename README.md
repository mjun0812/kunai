# kunai

My Python Package for Research and Utils.

## Install

```bash
pip install git+ssh://git@github.com/mjun0812/kunai.git

# upgrade
pip install git+ssh://git@github.com/mjun0812/kunai.git -U

# editable
git clone https://github.com/mjun0812/kunai.git
cd kunai
pip install -e .

# develop
git clone https://github.com/mjun0812/kunai.git
cd kunai
poetry install
```
